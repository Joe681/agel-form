agel-form | 使用文档

基于 element-ui form 的二次封装加强组件

起步 →

极简的思想

面向对象, 参数集中处理, 以一个 from 对象来完成所有的操作

高度的灵活

支持 element-ui from 所有组件，也可搭配自定义组件，且不用担心版本升级问题

快捷的使用

数据配置，全局配置，集成了更方便好用的扩展功能

# 如此简单

<template>
  <agel-from v-model="from"></agel-from>
</template>
 
<script>
export default {
  data() {
    return {
      from: {},
    };
  },
};
</script>
