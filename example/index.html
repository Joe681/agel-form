# 介绍

agel-form (opens new window) 是基于 element-ui form (opens new window) 的二次封装，极简的思想，完全的数据驱动，拥有绝对的灵活性，可以帮助你更快速的开发！up! up! up! (opens new window) (opens new window)

# 特性

该组件的思想就是以一个 form 对象来做所有的操作，它做了如下事情：

  • 支持 element-ui 所有表单组件 attributes 配置
  • 支持 element-ui 所有表单组件 events 事件
  • 支持 element-ui 所有表单组件 slots 插槽
  • 支持 element-ui Row Col 组件属性
  • 智能回填 form.data，placeholder，rules
  • 灵活自定义组件 / 搭配第三方组件
  • 全局配置
  • 联动显示隐藏禁用
  • 响应式布局

内置单组件，是基于 element-ui (opens new window) 的基础组件经过二次封装的单组件，在使用方式上同原组件保持一致，没有任何的破坏性，并且额外添加了扩展功能，完完全全可以代替原组件进行开发使用。

  • agel-select (opens new window)
